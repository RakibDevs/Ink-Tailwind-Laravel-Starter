To clone this repository -
```bash
git clone https://github.com/rakibdevs/Ink-Tailwind-Laravel-Starter.git

cd Ink-Tailwind-Laravel-Starter

composer update

cp .env.example .env

php artisan key:generate

npm install

npm run dev

php artisan migrate

php artisan serve
```
